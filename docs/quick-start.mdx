## Create Your Account

<Steps>
  <Step>
    ### Sign Up
    
    Visit [developers.brainus.lk](https://developers.brainus.lk) and create your free account. No credit card required.
  </Step>

{" "}

<Step>
  ### Generate API Key Navigate to Dashboard → API Keys and click "Generate New
  Key". Copy it immediately - you won't see it again!
</Step>

  <Step>
    ### Make Your First Request
    
    Test your API key with a simple curl command:

    ```bash
    curl -X POST "https://api.brainus.lk/api/v1/query" \
      -H "X-API-Key: $BRAINUS_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "query": "What is photosynthesis?",
        "store_id": "default"
      }'
    ```

    Prefer a visual interface? Check out the [API Playground](/docs/api-playground) to test queries interactively.

  </Step>

  <Step>
    ### Install SDK (Optional)
    
    For Python or JavaScript projects, install our official SDK:

    ```bash
    # Python
    pip install brainus-ai

    # JavaScript/TypeScript
    npm install @brainus/ai
    ```

  </Step>
</Steps>

## Quick Example

Here's how simple it is to query BrainUs API:

<Tabs items={['Python', 'JavaScript', 'cURL']}>
  <Tab value="Python">
    ```python
    from brainus_ai import BrainusAI

    async with BrainusAI(api_key="your-api-key") as client:
        result = await client.query(
            query="Explain photosynthesis",
            store_id="default"
        )
        print(result.answer)
        print(result.citations)
    ```

  </Tab>
  <Tab value="JavaScript">
    ```javascript
    import { BrainusAI } from '@brainus/ai';

    const client = new BrainusAI({ apiKey: 'your-api-key' });
    const result = await client.query({
      query: 'Explain photosynthesis',
      storeId: 'default'
    });
    console.log(result.answer);
    console.log(result.citations);
    ```

  </Tab>
  <Tab value="cURL">
    ```bash
    curl -X POST "https://api.brainus.lk/api/v1/query" \
      -H "X-API-Key: your-api-key" \
      -H "Content-Type: application/json" \
      -d '{"query": "Explain photosynthesis", "store_id": "default"}'
    ```
  </Tab>
</Tabs>
