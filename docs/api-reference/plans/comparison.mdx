## Free Plan

Perfect for testing, development, and small projects.

**Best for:**

- Testing and development
- Personal projects
- POCs and demos
- Learning the API

**Includes:**

- 300 requests/month
- 10 requests/minute
- 300 requests/day
- 3 API keys
- Basic analytics
- Community support

## Pro Plan

For production applications and growing businesses.

**Best for:**

- Production applications
- Growing user base
- Multiple projects
- Team collaboration

**Includes:**

- 10,000 requests/month
- 60 requests/minute
- 10,000 requests/day
- 10 API keys
- Advanced analytics
- Priority email support
- Webhook notifications

**Popular features:**

- Real-time usage alerts
- Detailed response time metrics
- API key usage breakdown
- Export usage data to CSV

## Enterprise Plan

For high-volume applications with dedicated support.

**Best for:**

- Enterprise applications
- High-volume traffic
- Mission-critical systems
- Custom requirements

**Includes:**

- 50,000 requests/month
- 200 requests/minute
- 50,000 requests/day
- Unlimited API keys
- Custom analytics dashboard
- Dedicated support engineer
- 99.9% uptime SLA
- Custom integrations
- Volume discounts

**Enterprise features:**

- Custom rate limits
- Dedicated infrastructure
- Priority feature requests
- Quarterly business reviews
- Custom SLA terms

<Callout type="tip">
  All plans include the same API features and access - only rate limits and
  support differ.
</Callout>

## Feature Details

### Analytics

**Basic Analytics (Free):**

- Request counts
- Success/failure rates
- Daily usage charts

**Advanced Analytics (Pro):**

- Hourly granularity
- Response time metrics
- API key breakdown
- Export to CSV
- 90-day history

**Custom Analytics (Enterprise):**

- Real-time monitoring
- Custom dashboards
- Integration with monitoring tools
- Unlimited history
- Custom reports

### Support

**Community Support (Free):**

- Discord community
- Documentation
- Email support (best effort)

**Priority Email (Pro):**

- 24-hour response time
- Technical support
- Integration assistance

**Dedicated Support (Enterprise):**

- Named support engineer
- 4-hour response time
- Phone support
- Custom integration help
- Quarterly reviews
